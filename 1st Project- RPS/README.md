Play against the computer in this classic hand game right in your terminal.

## Features

- Classic rules: **Rock** beats **Scissors**, **Scissors** beats **Paper**, **Paper** beats **Rock**
- Computer makes random choices
- Keeps score (player vs computer)
- Play multiple rounds
- Simple and beginner-friendly code

## How to Play

1. Run the script
2. Type `rock`, `paper`, or `scissors` (case-insensitive)
3. Or use shortcuts: `r`, `p`, `s`
4. Type `quit`, `q` or `exit` to stop playing

## Requirements

- Python 3.x (any recent version should work)

No external libraries needed! Just standard Python.
